# Kaggle Workspace

Kaggleコンペティションおよび機械学習の学習用ワークスペースです。

## 📁 ディレクトリ構成

```
kaggle-workspace/
├── Kaggle_training/       # 学習・練習用
│   ├── onehot-lesson3-engineering-randomforest.ipynb
│   ├── Titani_LogisticRegression.ipynb
│   └── udemy-titanic-lesson1 (1).ipynb
├── gpu_test.ipynb         # GPU動作確認用
├── test.ipynb             # テスト用ノートブック
└── test.py                # テスト用スクリプト
```

## 📚 学習コンテンツ

### Kaggle_training
- **Titanic**: ロジスティック回帰を用いた生存予測の練習
- **特徴量エンジニアリング**: ランダムフォレストとOne-Hotエンコーディング

## 🚀 セットアップ

このプロジェクトはDev Containerを使用しています。
VSCodeでフォルダを開くと、自動的に必要な環境が構築されます。
